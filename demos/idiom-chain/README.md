# 成语接龙

一个经典的成语接龙小游戏，玩家和电脑轮流接龙成语。

## 游戏规则

1. **轮流接龙**：玩家和电脑轮流说出成语
2. **首尾相接**：下一个成语的首字必须是上一个成语的尾字
3. **不能重复**：已经使用过的成语不能再次使用
4. **有效成语**：必须是4个字且在成语库中的成语
5. **胜负判定**：如果接不上来或输入错误，该方失败
6. **跳过功能**：玩家可以选择跳过，让电脑接龙

## 游戏模式

- **开始游戏**：玩家输入第一个成语开始游戏
- **随机开始**：系统随机选择一个成语，电脑先手

## 技术实现

- 纯 JavaScript 实现，无需依赖
- 包含约500个常用成语的数据库
- 智能 AI 接龙算法
- 实时验证和反馈

## 运行方式

直接在浏览器中打开 `index.html` 文件即可开始游戏。
