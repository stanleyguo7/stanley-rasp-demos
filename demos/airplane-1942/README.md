# 1942 飞机大战

一个使用 Three.js 制作的经典飞机大战游戏，类似 1942 游戏风格。

## 游戏说明

### 控制方式
- **WASD** 或 **方向键**: 控制飞机上下左右移动
- **空格键**: 发射子弹

### 游戏规则
- 控制你的绿色飞机躲避红色敌机和子弹
- 发射子弹击毁敌机获得分数
- 每 100 分升一级，难度逐渐增加
- 你有 3 条生命，被击中或碰撞会失去生命
- 生命耗尽游戏结束

### 游戏特性
- 敌机会自动生成并向下移动
- 敌机会随机发射子弹
- 随着等级提升，敌机生成速度加快，移动速度增加
- 实时显示分数、生命值和等级

## 技术实现

- **Three.js**: 3D 渲染引擎
- **WebGL**: 硬件加速渲染
- 碰撞检测系统
- 动态难度调整

## 运行方式

直接在浏览器中打开 `index.html` 文件即可开始游戏。
